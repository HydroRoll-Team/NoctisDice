# 内置指令

## .mode

::: details 别名
`.m`
:::

切换和查询跑团模式

## .set

::: details 别名
`.st`
:::

角色卡设定。

### `show`选项

::: tip
指令`.st show`是指令`.show`的别名，它的设立是为了适配通用的指令体系。
:::

显示角色卡信息。

### `del`选项

::: tip
指令`.st del`是指令`.del`的别名，它的设立是为了适配通用的指令体系。
:::

删除角色卡。

::: tip 示例

```bash
.st del card
```

:::

### `clear`选项

清空所有角色卡信息。

::: tip 示例

```bash
.st clear
```

:::
